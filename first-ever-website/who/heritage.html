An open letter to Mr Geoff Dawkins about the use of false filler text in academic coding projects.

Dear Mr Dawkins,

In the recent project you assigned us for the computer science 061 module entitled 'Fundamental Technologies for the World Wide Web', it was specified that 'the text should be your own (you are no allowed to simply copy text)'. When asked to clarify this, you insisted that we were not allowed to use any sort of text that wasn't original content written specifically for the website.

It was specifically noted that we were not to use the industry standard stock filler text known as Lorem Ipsum.

Lorem Ipsum is dummy Latin text that has been used by the printing and typesetting industry since the 1500s and has now made its way into the web development community. Lorem Ipsum has its roots in extracts from a book written by Cicero in around 45BC called 'The Extremes of Good and Evil'.

Lorem Ipsum is used instead of sensible text when show casing the design of a website. It is used as it has a relatively normal distribution of letters and words while not having any meaning to distract from the website itself.

This is the primary reason that I believe we should be able to use Lorem Ipsum text for the purpose of academic website construction. The content of the website does not serve any purpose, there are no marks to be gained for good content, however there does need to be some form of content in order to showcase the effects that we have chosen to implement though the use of HTML and cascading style sheets. Due to not being able to use Lorem Ipsum, I have had to spend a lot more time writing rubbish content, such as this open letter, instead of polishing my code.

Lorem Ipsum is not copyrighted, it is free to use, so it is not possible for any potential problems to arise from that. Lorem Ipsum text would probably set off the plagiarism alarms however this is the only problem that I can think of.

I believe that if you allowed the use of Lorem Ipsum on web projects, such as this one, you would allow students to dedicate more time to the important parts of the project and you might even see a noticeable improvement in the quality of the work as students focus shifted from content creation to coding.

Yours Sincerely,

Phil Reynolds
